Definition, relations and operations between sets

Definition of sets 1.1

Page: 1

Summary: A formal definition of a set, notation, elements and simple examples (finite sets, infinite sets, empty set).

Content: Introduces the membership relation (∈), subsets, and representations by extension and comprehension. Examples: A = {1,2,3}, B = {x ∈ ℕ | x ≤ 5}.

Sample exercises:

  • Give three examples of finite sets and two examples of infinite sets.
  • Write in extension and in comprehension: the set of even numbers ≤ 10.

Relations between sets 1.2

Page: 4

Summary: Inclusion, equality of sets, union, intersection, difference and complement.

Content: Formal definitions and properties (associativity, commutativity, distributivity). Venn diagrams to visualize operations.

Sample exercises:

  • Let A = {1,2,3} and B = {2,3,4}. Compute A ∪ B, A ∩ B and A \ B.
  • Check whether A ⊆ B for A = {1,2} and B = {1,2,3}.

Operations between sets 1.3

Page: 7

Summary: Set operations with applied exercises, mind maps and practical examples.

Content: Use of operations to solve problems, relationship between operations and set algebra. Examples and counterexamples.

Sample exercises:

  • Find the result of the operation: (A ∪ B) ∩ C for A={1,2}, B={2,3}, C={2}.
  • Illustrate by a Venn diagram the operation (A \ B) ∪ (B \ A).

Self-assessment test 1 Self-test

Page: 10

Self-assessment test: A set of 6 questions covering the key notions of the chapter.

  1. Define a set and give an example.
  2. What is the complement of A in the universe U?
  3. Compute A ∪ B and A ∩ B for A={1,3,5} and B={2,3,4}.
  4. Give an example of disjoint sets.
  5. True or false: if A ⊆ B and B ⊆ A then A = B.
  6. Represent A ∪ B using a Venn diagram (description).

Suggested answers are available in the appendix.
