1.1 Définition des ensembles

Définition 1

Nous appelons éléments les objets étudiés. Un ensemble est une collection bien définie d'éléments distincts.

Exemples :

  • {pomme, banane, pastèque} est un ensemble. La pomme, la banane et la pastèque sont des éléments de cet ensemble.
  • {x | -1 < x < 3} est aussi un ensemble. Les nombres comme -½, 0, ⅔, 1 sont tous des éléments de cet ensemble.

Caractéristiques d'un ensemble

  1. Détermination : Les éléments d'un ensemble sont clairement définis.
  2. Distinction : Les éléments d'un ensemble sont tous différents.
  3. Non-ordre : Les éléments d'un ensemble n'ont pas d'ordre spécifique.

Notation

Nous utilisons généralement les lettres majuscules A, B, C, ... pour désigner les ensembles et les lettres minuscules a, b, c, ... pour désigner les éléments.

Ensembles de nombres usuels

ℕ : Ensemble des nombres naturels
ℤ : Ensemble des entiers
ℤ⁺ : Ensemble des entiers positifs
ℚ : Ensemble des rationnels
ℝ : Ensemble des réels

Relation entre les ensembles de nombres :

ℝ = ℚ ∪ {nombres irrationnels}
ℚ = {nombres pouvant s'écrire sous forme de fraction}
ℤ = {..., -3, -2, -1, 0, 1, 2, 3, ...}
ℕ = {0, 1, 2, 3, ...}

Relation d'appartenance

Si a est un élément de l'ensemble A, on dit que a appartient à A, noté a ∈ A.

Si a n'est pas un élément de A, on dit que a n'appartient pas à A, noté a ∉ A.

1.2 Relations entre ensembles

Égalité d'ensembles

Deux ensembles A et B sont égaux s'ils contiennent exactement les mêmes éléments, noté A = B.

Exemple :

{-1, 3, 6} = {3, -1, 6} = {6, -1, 3}

Sous-ensemble

Un ensemble A est un sous-ensemble d'un ensemble B si tous les éléments de A sont aussi des éléments de B, noté A ⊆ B.

Sous-ensemble propre

Si A ⊆ B et A ≠ B, alors A est un sous-ensemble propre de B, noté A ⊂ B ou A ⊊ B.

L'ensemble vide

L'ensemble vide, noté ∅ ou {}, est l'ensemble qui ne contient aucun élément.

Propriété importante : L'ensemble vide est un sous-ensemble de tout ensemble : ∅ ⊆ A pour tout ensemble A.

Nombre de sous-ensembles

Un ensemble à n éléments possède 2^n sous-ensembles et 2^n - 1 sous-ensembles propres.

Exemple :

L'ensemble {a, b} a 4 sous-ensembles : ∅, {a}, {b}, {a,b}.

Il a 3 sous-ensembles propres : ∅, {a}, {b}.

1.3 Opérations sur les ensembles

Union d'ensembles

L'union de deux ensembles A et B, notée A ∪ B, est l'ensemble des éléments qui appartiennent à A ou à B (ou aux deux).

A ∪ B = {x | x ∈ A ou x ∈ B}

Intersection d'ensembles

L'intersection de deux ensembles A et B, notée A ∩ B, est l'ensemble des éléments qui appartiennent à la fois à A et à B.

A ∩ B = {x | x ∈ A et x ∈ B}

Complémentaire d'un ensemble

Étant donné un ensemble universel U, le complémentaire d'un ensemble A, noté ∁_U A ou Aᶜ, est l'ensemble des éléments de U qui n'appartiennent pas à A.

∁_U A = {x ∈ U | x ∉ A}

Les intervalles

Certains ensembles de nombres réels peuvent être représentés par des intervalles :

Type d'intervalle Notation Représentation ensembliste
Intervalle fermé [a, b] {x | a ≤ x ≤ b}
Intervalle ouvert (a, b) {x | a < x < b}
Intervalle semi-ouvert à gauche (a, b] {x | a < x ≤ b}
Intervalle semi-ouvert à droite [a, b) {x | a ≤ x < b}

Exercices du chapitre

Exercice 1.1

Utilisez les symboles ∈ ou ∉ pour compléter :

  1. 5 __ ℕ, -3 __ ℕ
  2. 8 __ ℤ, 3/2 __ ℤ
  3. 3.14 __ ℚ, √3 __ ℚ
  4. 2 __ ℝ, π/5 __ ℝ
Exercice 1.2

Écrivez tous les sous-ensembles de l'ensemble {1, 3, 5, 7}. Indiquez lesquels sont des sous-ensembles propres.

Exercice 1.3

Soient A = {-1, 1, 3, 5} et B = {-1, 2, 4, 6}. Calculez :

  1. A ∪ B
  2. A ∩ B
Exercice 1.4

Soit U = {0, 2, 4, 6, 8, 10}, A = {2, 6, 8}, B = {0, 4, 6, 10}. Calculez :

  1. ∁_U A
  2. ∁_U B
  3. (∁_U A) ∩ (∁_U B)

Résumé du chapitre

Concepts clés

  • Ensemble : Collection d'éléments distincts et bien définis
  • Notations : ℕ, ℤ, ℚ, ℝ pour les ensembles de nombres
  • Appartenance : a ∈ A (a appartient à A), a ∉ A (a n'appartient pas à A)
